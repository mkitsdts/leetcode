给你一个整数数组 nums ，判断是否存在三元组 [nums[i], nums[j], nums[k]] 满足 i != j、i != k 且 j != k ，同时还满足 nums[i] + nums[j] + nums[k] == 0 。
请你返回所有和为 0 且不重复的三元组。
注意：答案中不可以包含重复的三元组。

示例 1：
输入：nums = [-1,0,1,2,-1,-4]
输出：[[-1,-1,2],[-1,0,1]]
解释：
nums[0] + nums[1] + nums[2] = (-1) + 0 + 1 = 0 。
nums[1] + nums[2] + nums[4] = 0 + 1 + (-1) = 0 。
nums[0] + nums[3] + nums[4] = (-1) + 2 + (-1) = 0 。
不同的三元组是 [-1,0,1] 和 [-1,-1,2] 。
注意，输出的顺序和三元组的顺序并不重要。

示例 2：
输入：nums = [0,1,1]
输出：[]
解释：唯一可能的三元组和不为 0 。

示例 3：
输入：nums = [0,0,0]
输出：[[0,0,0]]
解释：唯一可能的三元组和为 0 。

提示：
3 <= nums.length <= 3000
-10^5 <= nums[i] <= 10^5

遇到数组题第一步是尝试排序，排序好之后看有没有思路。一般题在排序好之后思路会变得清晰。

实例的nums = [-1,0,1,2,-1,-4]排好序后变成[-4,-1,-1,0,1,2]

从第一个数-4开始，我们定义left等于下一位数，right等于最后一位数，然后开始搜索。如果和小于0，就把左指针向右移动，如果是相同的数就继续右移；如果和大于0，就把右指针向左移动，如果相同的数就继续左移。搜索这个循环直到两个指针重合。

下一次搜索时，我们可以判断一下是否与上一个数相等，如果相等就没必要再次搜索了。除此之外，我们也可以判断一下是否正数，是正数也没必要再搜索了，因为搜索的值作为最小值都是正数，加上更大的数肯定不可能等于0。

以上这些条件足以解决问题。
